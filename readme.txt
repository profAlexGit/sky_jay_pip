Все изменения в проекте

1. В папке data создался файл models.py
    Где хранятся все модели таблиц БД и методы для создания их экземпляров
1.1. класс User
    id - Integer. Идентификационный ключ к пользователю
    isUser - Boolean. Определяет психолога от пользователя. Если True то пользователь, иначе психолог.
    name - String. Имя пользователя
    surname - String. Фамилия пользователя
    dateBirth - String но скоро изменю на DateTime. Дата рождения
    placeResidence - String. Место проживания (не обязательно)
    password - BLOB (bytes). Пароль пользователя (зашифрованная строка с помощью sha256)
    email - String. Email пользователя

1.2. Метод createUserModel(_isUser: bool = True)
    _isUser - bool. Необязательный параметр но он подставляется в User.isUser

    Функция получающая из формы регистрации сайта все данные для создания и возращения
    экземпляра класса User

1.3. Метод spamUsers(stop: int = 1000, _start: int = 0)
    Функция создающая множество фейковых пользователей

2. Изменения в файле main.py
    Импортирован модуль data/models.py
2.1. Добавлен метод clearDB()
    Метод который полностью очищает всю БД от пользователей
2.2. В @app.route('/')
    Заменена страница возвращающаяся в виде ответа пользователю
2.3. Добавлена обработка ссылки "http://127.0.0.1:8000/all-users"
    Страница выводящая всех пользователей и психологов
2.4. Добавлена обработка ссылки "http://127.0.0.1:8000/registration"
    Просто возвращает страницу с переходом на страницы с регистрацией психолога или пользователя
2.5. Добавлена обработка ссылки "http://127.0.0.1:8000/psycho-registration"
    Принимает POST и GET запросы для регистрации психолога
2.6. Добавлена обработка ссылки "http://127.0.0.1:8000/user-registration"
    Принимает POST и GET запросы для регистрации психолога

3. Изменения в папке templates
    Добавлены страницы
3.1. templates/index.html
    Страница с тремя ссылками
    1. Регистрация ("http://127.0.0.1:8000/registration")
    2. Вход ("http://127.0.0.1:8000/login") (Не обработана. Выдаёт код 404)
    3. Все пользователи ("http://127.0.0.1:8000/all-users")
3.2. templates/all-users.html
    Страница со списком всех пользователей и психологов
3.3. templates/registration_page1.html
    Страница выбора кем пользователь будет зарегистрирован (пользователь или психолог)
3.4. templates/registration_page2.html
    Страница регистрации пользователя или психолога (в зависимости от того на какую ссылку нажал пользователь на
                                                     этапе 3.3)
    После нажатия на кнопку "Зарегистрироваться" перенаправляется на страницу "http://127.0.0.1:8000/all-users"
    в будущем эта ссылка будет изменена на "http://127.0.0.1:8000/login"
